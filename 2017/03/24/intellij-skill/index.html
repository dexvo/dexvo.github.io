Intellij idea使用技巧

Mar 24, 2017 | Hits

mac键盘符号和修饰说明

  1. ⌘ Command
  2. ⇧ Shift
  3. ⌥ Option
  4. ⌃ Control
  5. ↩︎ Return/Enter
  6. ⌫ Delete
  7. ⌦ 向前删除键（Fn+Delete）
  8. ⇞ Page Up（Fn+↑）
  9. ⇟ Page Down（Fn+↓）
    11.Home Fn + ←
    12.End Fn + →
    13.⇥ 右制表符（Tab键）
    14.⇤ 左制表符（Shift+Tab）
    15.⎋ Escape (Esc)

    快捷键(mac)

  10. command + E 查找最近访问的文件
  11. command + shift + E 访问最近编辑的文件
  12. double shift 输入 “/ + 文件夹名” 跳到指定文件夹
  13. command + shift + enter 补全改行的分号
  14. command + shift + A 执行某个action或者打开某个插件
  15. command + shift + V 访问历史粘贴板
  16. alt + enter 将一个字符串标记为某一种格式(json,正则)
  17. shift + f7 debug时可以选择进入当前行的某个方法
